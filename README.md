# Okami OS

A lightweight, keyboard-driven Linux distribution based on **Mageia**, with **Fluxbox** as the window manager and a **web-based desktop environment** built with HTML, CSS, and JavaScript.

## 🎯 Goals
- **Lightweight**: Minimal resource usage for optimal performance.
- **Keyboard-Driven**: Efficient workflows using keyboard shortcuts.
- **Web-Based DE**: A modern desktop experience powered by web technologies.
- **User-Friendly**: Accessible for both developers and everyday users.

## 📦 Installation
See [INSTALLATION.md](docs/INSTALLATION.md) for detailed instructions.

## 🛠️ Features
- Fluxbox as the default window manager.
- Web-based desktop environment (HTML/CSS/JS).
- Customizable themes and layouts.
- Keyboard shortcuts for efficiency.
